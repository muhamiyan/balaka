# Buku Besar (General Ledger)

## Pengertian

Buku Besar adalah laporan yang menampilkan mutasi dan saldo setiap akun berdasarkan jurnal yang tercatat dalam sistem.

## Mengakses Buku Besar

1. Klik menu **Buku Besar** di sidebar
2. Halaman akan menampilkan daftar jurnal dengan saldo berjalan

## Fitur Filter

| Filter | Fungsi |
|--------|--------|
| Akun | Pilih akun tertentu untuk dilihat |
| Tanggal Mulai | Awal periode yang ingin ditampilkan |
| Tanggal Selesai | Akhir periode yang ingin ditampilkan |
| Pencarian | Cari berdasarkan keterangan jurnal |

## Cara Menggunakan

1. Pilih **Akun** yang ingin dilihat dari dropdown
2. Tentukan **Periode** dengan memilih tanggal mulai dan selesai
3. Klik **Terapkan Filter**
4. Sistem akan menampilkan:
   - Ringkasan saldo (Opening, Debit, Kredit, Closing)
   - Daftar jurnal dengan saldo berjalan

## Memahami Tampilan

### Kartu Ringkasan

- **Saldo Awal** - Saldo akun di awal periode
- **Total Debit** - Jumlah semua transaksi debit dalam periode
- **Total Kredit** - Jumlah semua transaksi kredit dalam periode
- **Saldo Akhir** - Saldo akun di akhir periode

### Tabel Jurnal

| Kolom | Keterangan |
|-------|------------|
| Tanggal | Tanggal jurnal dicatat |
| No. Jurnal | Nomor referensi jurnal |
| Keterangan | Deskripsi transaksi |
| Debit | Jumlah di sisi debit |
| Kredit | Jumlah di sisi kredit |
| Saldo | Saldo berjalan setelah transaksi |

## Melihat Detail Jurnal

1. Klik nomor jurnal di tabel
2. Halaman detail menampilkan:
   - Semua baris jurnal (akun yang terlibat)
   - Dampak ke setiap akun
   - Link ke transaksi sumber

## Saldo Berjalan (Running Balance)

Saldo berjalan menunjukkan saldo akun setelah setiap transaksi dicatat. Ini membantu Anda:

- Melacak pergerakan saldo akun
- Mengidentifikasi transaksi yang menyebabkan perubahan signifikan
- Memverifikasi kebenaran pencatatan

## Ekspor Data

Untuk mengekspor data buku besar:

1. Terapkan filter yang diinginkan
2. Klik tombol **Ekspor**
3. Pilih format: Excel atau PDF
4. File akan diunduh ke komputer Anda

## Tips Penggunaan

1. Rekonsiliasi saldo bank secara berkala dengan buku besar
2. Periksa akun-akun penting di akhir bulan
3. Gunakan filter periode untuk analisis tren
4. Ekspor data untuk dokumentasi dan audit
