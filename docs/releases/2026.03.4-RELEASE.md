# Release 2026.03.4

## Highlights

Code quality release: 54 SonarCloud issues resolved to zero. CSP-compliant event handlers. Reusable period selector fragment across all report pages. API hardening for invalid input.

No schema migration changes. No breaking API changes.

## What's New

### Reusable Period Selector Fragment

- Period selector extracted into a shared Thymeleaf fragment, reusable across all report pages
- Reports index page reorganized with period date capping to today
- Active period preset button now highlighted on page load

## Improvements

- 54 SonarCloud issues resolved (reliability, maintainability, code smells) — now at zero
- Inline `onclick` handlers replaced with event delegation for Content Security Policy compliance
- Balaka brand assets unified — full logo SVGs, brand guidelines, consistent proportions across theme and user manual
- Period-selector fragment HTML wrapper now includes `lang` and `title` attributes
- Test locators scoped to `#period-report-content` for more robust Playwright tests

## Bug Fixes

- `NumberFormatException` in `parseYearMonth` API method now returns proper error response instead of 500
- Period report page crash caused by Thymeleaf `#strings.padLeft` type mismatch
- Full logo SVG `viewBox` widened from 720 to 850 to prevent clipping

## Breaking Changes

- None

## Dependencies
